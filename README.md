# Godot Mobile Game Demo
Simple 3D mobile game based on Godot 4.
Contains basic player movement logic for Android.

## Features
- 3D character movement
- Mobile virtual joystick support
- Ready to build Android APK

## How to use
1. Download all files
2. Import project into Godot 4
3. Build your 3D scene manually
4. Run or export to APK

## Input
- Mobile: Virtual joystick
- PC: W A S D
